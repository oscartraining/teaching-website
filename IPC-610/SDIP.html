📘 詳細異常說明

❌ 錫多 Excessive Solder

說明： 錫料堆積超出焊墊範圍，可能短路或與其他腳橋接。

常見原因： 錫膏印刷過多、爐溫不穩、元件偏移導致焊料集中。

Class 2 標準： 若無短路、仍包覆良好且不影響功能，可接受。

❌ 錫少 Insufficient Solder

說明： 焊點錫料不足，導致腳位未完全包覆、機械強度或導通不足。

常見原因： 印刷厚度不均、元件位置偏移、加熱不足。

Class 2 標準： 若影響導通或固定，即判定為 Defect。

❌ 錫橋 Solder Bridge

說明： 相鄰焊點間錫料連接，形成短路。

常見原因： 錫膏印刷過量、貼片偏移、焊料回流流動不均。

Class 2 標準： 造成短路即為 Defect。

❌ 錫洞 Solder Void

說明： 焊點內部氣泡形成空洞，影響熱導與機械強度。

常見原因： 焊膏潮濕、加熱不完全、預熱不足。

Class 2 標準： 可容許小空洞，但不能影響結構與導通。

❌ 錫裂 Solder Crack

說明： 焊點形成後產生裂縫，易造成溫升斷點。

常見原因： 溫度變化劇烈、材料膨脹不一致、焊料品質不良。

Class 2 標準： 任何影響導通的裂縫皆為 Defect。
